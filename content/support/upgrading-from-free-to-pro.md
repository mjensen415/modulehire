# Upgrading from Free to Pro

The free plan gives you access to the core ModuleHire workflow — upload a resume, build your module library, and generate up to 2 tailored resumes per month. Pro removes those limits and unlocks the full experience.

## What's included in each plan

| Feature | Free | Pro |
|---|---|---|
| Resume upload & module parsing | ✓ | ✓ |
| Module library | ✓ | ✓ |
| Tailored resume generations | 2/month | Unlimited |
| ATS Estimator | Score only | Full breakdown |
| Module editing | Read-only after parse | Full editing |
| Multiple base resume uploads | — | ✓ |
| Priority support | — | ✓ |

## How to upgrade

1. Go to **Account Settings** from the top navigation
2. Click **Upgrade to Pro**
3. Choose **Monthly ($19/mo)** or **Annual ($99/yr)**
4. Enter your payment details and confirm

Your account upgrades immediately — no waiting period.

## Not ready for a subscription?

If you only need a resume or two right now, single-purchase options are available:

- **Single resume** — $9
- **5-pack** — $29

These don't renew and don't require a subscription. You can add more credits anytime from your account page.
